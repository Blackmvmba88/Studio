# Workflow Memory

## Default Creative Pipeline

1. Capture idea or prompt.
2. Generate lyrics or concept.
3. Generate audio.
4. Create cover art.
5. Prepare metadata.
6. Publish to SoundCloud / DistroKid / Pinterest.
7. Save what worked.
8. Turn repeated steps into scripts.

## Rules

- Do not work random.
- Save reusable prompts.
- Keep private data out of public files.
- Prefer local AI for repeatable drafting.
- Use premium AI for strategy, hard reasoning, images, and browser actions.
- Every serious project gets README, roadmap, and next commits.
